# SQL Ecommerce Sales Analysis

## Project Overview
This project analyzes ecommerce sales data using SQL to identify revenue trends, top-performing categories, customer behavior, and country-wise sales performance.

## Tools Used
- MySQL
- Excel / CSV
- GitHub

## Dataset
The dataset contains:
- orders data
- users data
- products data

## Analysis Performed
- Total Revenue
- Total Orders
- Average Order Value
- Sales by Category
- Sales by Country
- Yearly Sales Trend
- Top Customers

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- JOINS
- Aggregate Functions
- Window Functions

## Key Insights
- Beauty category generated the highest revenue.
- Electronics category contributed very low sales.
- Revenue is concentrated in a few key countries.
- Average order value is moderate, showing small basket size.

## Recommendations
- Improve low-performing categories through pricing and promotions.
- Increase average order value using upselling and bundle offers.
- Focus marketing on top-performing countries while improving weak regions.
